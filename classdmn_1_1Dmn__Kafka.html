Detailed Description

Apache Kafka producer/consumer that implements the Dmn_Io<std::string> interface.

Wrap librdkafka so that Kafka topics can be used as drop-in Dmn_Io endpoints. Construct with Role::kProducer to send messages or Role::kConsumer to receive them. Both directions use the same configuration map; Dmn_Kafka-specific keys (Topic, Key, PollTimeoutMs) are extracted before the remainder is forwarded to rd_kafka_conf_set().
