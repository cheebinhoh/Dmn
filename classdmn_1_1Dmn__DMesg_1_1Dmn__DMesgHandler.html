Detailed Description

Dmn_DMesgHandler is an IO-style interface for clients to publish and consume dmn::DMesgPb messages. It implements Dmn_Io<DMesgPb>.

The handler composes a small nested subscriber object (Dmn_DMesgHandlerSub) that integrates with the underlying Dmn_Pub infrastructure. The handler maintains its own filters, async processing callback, per-topic running counters, and a small buffer for arriving messages.

Clients obtain handlers from Dmn_DMesg::openHandler(...) and release them via Dmn_DMesg::closeHandler(...).
