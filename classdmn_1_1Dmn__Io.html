Read and return the next available item.

The call blocks until data is available. Returns std::nullopt to signal end-of-stream (e.g. EOF, closed pipe, or unrecoverable error); callers should treat std::nullopt as a permanent stop condition and cease further reads.

Returns
optional<T> containing the next item, or std::nullopt on end-of-stream.

Read up to count items, optionally waiting up to timeout microseconds.

The default implementation returns an empty vector. Concrete subclasses may override this to provide bulk-read semantics consistent with Dmn_BlockingQueue::pop(count, timeout).

Parameters
countMaximum number of items to return (must be > 0).
timeoutMaximum wait time in microseconds; 0 means wait indefinitely.
Returns
Vector of up to count items (possibly fewer on timeout).
