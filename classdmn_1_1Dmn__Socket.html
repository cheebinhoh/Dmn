Detailed Description

Implements a socket-based Dmn_Io for std::string messages.

Provides a synchronous socket interface that conforms to the Dmn_Io<std::string> contract. The class manages a single file descriptor (m_fd) representing a connected TCP socket to the specified IPv4 address and port.

Thread-safety: Instances are NOT inherently thread-safe. Synchronize access externally if multiple threads share an instance.

Lifetime/ownership: The socket file descriptor is owned by the object and closed in the destructor. Copy and move operations are deleted to avoid accidental sharing of the descriptor.
