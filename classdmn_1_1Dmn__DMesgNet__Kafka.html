Dmn_Kafka::ConfigType  configs

Construct a Dmn_DMesgNet_Kafka by creating a Kafka consumer and producer from configs and wiring them as the input and output handlers of the underlying Dmn_DMesgNet.

The caller must supply the common broker connection parameters in configs (e.g. bootstrap.servers, SASL credentials). The following keys are set internally and must NOT be present in configs:

  • "group.id" : set to name.
  • "auto.offset.reset": set to "earliest".
  • "acks" : set to "all".
  • Dmn_Kafka::Topic : set to "Dmn_dmesgnet".
  • Dmn_Kafka::Key : set to "Dmn_dmesgnet".
Parameters
nameIdentifier used for both the Dmn_DMesgNet instance and the Kafka consumer group ID.
configsKafka configuration entries (see above for reserved keys).
