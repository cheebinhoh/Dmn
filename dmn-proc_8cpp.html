Detailed Description

Lightweight RAII wrapper around native pthread functionality.

Copyright © 2025 Chee Bin HOH. All rights reserved.

This module provides a C++ wrapper around POSIX threads (pthread) with RAII semantics. It manages thread lifecycle and ensures proper cleanup on destruction.

void dmn::cleanupFuncToUnlockPthreadMutex ( void *  arg)

Helper function to unlock a pthread mutex during thread cancellation.

Cleanup function used with pthread_cleanup_push/pop. Expects a pointer to a pthread_mutex_t and unlocks it.

This function is declared here so it can be used with the macros above and with pthread_cleanup_push/pop in implementation files.

Parameters
argPointer to a pthread_mutex_t to be unlocked (void* per pthread API)

This is used as a cleanup handler registered with pthread_cleanup_push to ensure the mutex is properly released when a thread is cancelled.

argPointer to the pthread_mutex_t to unlock
