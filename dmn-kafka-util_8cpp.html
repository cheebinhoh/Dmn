Detailed Description

Implementation of the librdkafka configuration helper.

Copyright © 2025 Chee Bin HOH. All rights reserved.

Provides set_config(), a thin wrapper around rd_kafka_conf_set() that converts the C-style output-parameter error string into a std::expected return value for idiomatic C++ error handling.
