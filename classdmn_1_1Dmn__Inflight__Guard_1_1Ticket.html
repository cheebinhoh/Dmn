Detailed Description

template<class T = std::monostate>
class dmn::Dmn_Inflight_Guard< T >::Ticket

RAII guard that represents one active in-flight operation.

Constructing a Ticket increments the owning guard's in-flight counter and invokes enterInflightGuardFnc() on the guard. Destroying the ticket decrements the counter and invokes leaveInflightGuardFnc().

Construction throws std::runtime_error if the guard is already closed (i.e., shutdown has begun).

Template Parameters
TOptional per-call payload type stored inside the ticket (defaults to std::monostate for no payload).

Release the in-flight ticket.

Invokes leaveInflightGuardFnc() on the owning guard (swallowing any exceptions to satisfy the noexcept contract), then decrements the in-flight counter and notifies any threads waiting in waitForEmptyInflight().
