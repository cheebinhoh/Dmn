Detailed Description

Custom final awaiter that resumes the registered continuation.

When the coroutine body finishes (or returns), final_suspend() returns this awaiter. await_suspend() uses symmetric transfer to resume the coroutine that is awaiting this task (stored in m_continuation), which avoids unbounded stack growth from recursive resumes.
